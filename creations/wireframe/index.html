Layout

constitution du layout pour ce site je l'ai fait en utilisant que les flex et en utilisant le display grid je vais detailler les 2 methodes et vous allez conclure qu'en utlisant les grid on vas plus vite 

          <div class="grid-layout">
          <header></header>
          <div class="picture"></div>
          <aside></aside>
          <main></main>    
          </div>     
        
          .grid-layout{
            flex: 1;
            min-height: 30vh;
            background: var(--blue-light);
            display: grid;
            grid-template-columns: 1fr 1fr 1fr;
            grid-template-rows: 5rem 1fr;
            grid-template-areas:
            "header header picture"
            "main main sidebar";
            header{
              grid-area: header;
              background: var(--red);
            }
            .picture{
              grid-area: picture;
              background: var(--pink);
            }
            aside{
              grid-area: aside;
              background: var(--gray);
            }
            main{
              grid-area: main;
              background: var(--green);
            } 
          }
