Mise en place de l'environnement

  • Installation de SASS

    sass est un preprocesseur css c'est un language qui sera ensuite compilé en css

    Pour utiliser sass il faut installer npm, et pour utiliser npm faut installer nodejs

    npm: gestionnaire de paquets officiel de nodejs, installer par defaut lorsque nodejs est installé,

    nodejs: est un environnement serveur permettant d'executer du javascript côté serveur.

    pour installer nodejs on peut utiliser un executable sur le site officiel de nodejs quelque soit le system d'exploitation

    On peut ensuite ouvrir un terminal et taper node -v pour vérifier que nodejs est bien installer et trouver la version installer

Créer un fichier package.json

  • Création d'un nouveau dossier berberes
  • npm init : cette commande créera un fichier package.json
  • package.json permet de :
    • lister ls paquets utiliser dans le projet : les dépendances
    • spécifie les versions de dépendance
    • Installer les dépendance du projet

Installation de la première dépendance: node-sass

npm i node-sass

ce paquet sera installés dans le dossier node_module et sera ajoutés comme dépendance dans le dossier package.json 

                {
                  "name": "CV",
                  "version": "1.0.0",
                  "description": "",
                  "main": "index.js",
                  "scripts": {
                    "sass": "node-sass -w scss -o css"
                  },
                  "keywords": [],
                  "author": "",
                  "license": "ISC",
                  "dependencies": {
                    "node-sass": "^4.14.1"
                  }
                }

au niveau de l'entrée script sur le fichier package.json on rajoute un script pour executer SASS "sass": "node-sass -w scss -o css"

  • -w:permet d'indiquer le dossier ou les fichiers contenant les fichiers sass qui vont etre suivie et coverti en css
  • -o: permet d'indiquer ou seront mis les fichiers css
  • npm run sass: pour lancer le script
  • ctrl + c: couper le script
